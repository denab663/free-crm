# DeepL API 文档翻译字符计算规则详解

## 文档翻译中的字符计费标准

在使用 DeepL API 进行文档翻译时，不同文件类型的字符计算规则有所差异：

### 按最低50,000字符计费的文件类型
以下文件类型即使实际字符数较少，系统也会按**最低50,000字符**计费：
- Microsoft Word (.docx/.doc)
- PowerPoint 演示文稿 (.pptx)
- Excel 表格 (.xlsx)
- PDF 文档 (.pdf)

### 按实际字符数精确计费的文件类型
以下文件类型将**精确计算实际字符数**，没有最低字符限制：
- 网页文件 (.html)
- 字幕文件 (.srt)
- 纯文本文件 (.txt)
- 本地化文件 (.xlf/.xliff)
